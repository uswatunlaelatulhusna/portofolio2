# Portofolio Sederhana (HTML, CSS, JS)

Ini contoh template portofolio sederhana yang dibuat dengan HTML, CSS, dan JavaScript vanilla. Cocok untuk dijadikan titik awal dan dikustomisasi.

Isi folder:
- index.html — halaman utama
- styles.css — styling responsif
- script.js — interaksi (nav mobile, form simulasi, modal, copy email)
- README.md — dokumentasi singkat

Cara pakai:
1. Simpan file-file di atas ke dalam satu folder.
2. Ganti teks [Nama Anda], email, tautan sosial, dan gambar placeholder sesuai kebutuhan.
3. Buka `index.html` di browser. Untuk pengembangan lebih lanjut, jalankan server lokal (mis. `npx http-server` atau Live Server extension).

Saran kustomisasi:
- Ganti placeholder gambar dengan foto asli (letakkan di folder `images/`).
- Tambahkan proyek nyata dengan tautan live/repo.
- Tambahkan form backend (mis. Netlify Forms, Formspree, atau API tersendiri) bila ingin menerima pesan nyata.
- Jika mau deploy: gunakan GitHub Pages, Netlify, atau Vercel.

Pertanyaan atau minta fitur tambahan (tema terang, animasi lebih kompleks, integrasi CMS), beri tahu saya dan saya bantu kembangkan.
